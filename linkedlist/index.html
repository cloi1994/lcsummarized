1. 单链表的创建

单链表的创建一般分为分为头插发(Head Insert)和尾插法(Tail Insert)

头插法是每次将新的节点插入到头部,这样得到的链表顺序是逆序

尾插法是每次将新的节点插入到尾部,这样得到的链表顺序是正序的. 值得注意的是,采用尾插法的时候需要一个尾指针,时时刻刻指向链表的尾部

2. 链表的操作

链表的插入无非是插入,删除操作.

在插入或删除的时候,一定要拿到他的Prev Pointer

为了便于链表的操作(例如,插入删除, 改变Head),一般使用带头结点的链表(Dummy Node)

常见的题目如下:

  • Add Two Numbers
  • Merge Two Sorted Lists
  • Partition List
  • Remove Linked List Elements

3. 链表最多使用三个Pointer

Prev - 用來Delete CurNode

Cur - 用來遍历Linklist

Next -保存CurNode的下一个Pointer,避免修改了Cur.next,而无法得到Next

常见的题目如下:

  • 61 Rotate List
  • 82 Remove Duplicates from Sorted List II
  • 91 Reverse Linked List I
  • 92 Reverse Linked List II

4. LinkedList的中点

使用两个指针slow,fast slow和fast同时向前跳, slow每次向前跳一个节点, fast每次向前跳两个节点, 当fast调到尾部的时候, slow就指向了中点

        while fast.next and fast.next.next:
            slow = slow.next
            fast = fast.next.next

5. LinkedList - Remove k from the end of the list

fast先前跳K步, slow和fast每次向前跳一个节点, 当fast调到尾部的时候, slow就指向了K Node的Prev Node.

6. LinkedList Cycle

a) 链表是否存在环?

我们可以使用两个指针slow和fast, fast每次前进两步,slow每次前进一次

如果fast和slow都到链表尾部且为空,则不在环.

如果fast和slow第一次相遇到某一节点,且不为空,则存在环.

        while fast.next and fast.next.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                return true
        return false

b) 链表环的入口节点?

根据第一次相遇的地方,把slow指回head,然后slow,fast在相同的速度再走一遍,当他们再次相遇就是入口了.

        while fast.next and fast.next.next:
            slow = slow.next
            fast = fast.next.next
            if fast == slow:
                slow = head
                while slow != fast:
                        slow = slow.next
                        fast = fast.next
            return slow
